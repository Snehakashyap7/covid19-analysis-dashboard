COVID-19 Analysis Dashboard (Power BI)
This project presents a Power BI dashboard analyzing the global impact of the COVID-19 pandemic. It uses real-world data to visualize trends in total cases, active cases, deaths, recoveries, and regional distribution.

📊 Dashboard Insights

- Total Cases: 1410M
- Total Deaths: 14M
- Active Cases: 159M
- Total Recovered: 1236M
- Recovery Rate: 87.7%
- Death Rate: 0.99%

🔸 Visualizations Included:

- Line chart of total cases, recoveries, and deaths
- Continent-wise and country-wise case distribution
- Interactive filters for region selection
- Map visualization for geographical insight

📁 Files Included

- `covid19_dashboard.pbix` – Power BI file
- `screenshots/` – Dashboard screenshot(s)

🧰 Tools Used
- Power BI
- DAX
- Power Query Editor

🔍 Use Case

This dashboard was created as a hands-on case study project to enhance real-time analytics, DAX measures, and geospatial data visualization skills using COVID-19 open data.
